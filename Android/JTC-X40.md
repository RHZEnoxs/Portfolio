大型顯示器 - JTC-X40
======
搭配大型顯示使用，提供藍芽連斷線 即時數值顯示功能，快速參數設定功能。

+ 獨立開發
+ 習得技能
    + Android Studio IDE
    + RecyclerView 佈局
    + TabLayout 佈局
    + 多國語系配置
+ 負責部分
    + 100 % Android
